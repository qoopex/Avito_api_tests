## Что тестируется

-   **Создание объявлений** - валидные и невалидные данные.
    
-   **Получение объявлений** - по ID и пользователю.
    
-   **Статистика** - просмотры, лайки, контакты.
    
-   **Удаление объявлений** - существующие и несуществующие.

## Основные компоненты

-   `APIclient.py` - HTTP клиент для API вызовов.
    
-   `conftest.py` - тестовые данные и конфигурации.
    
-   Файлы тестов - организованы по функциональности.

## Отчеты тестирования

-   Смотрите `BUGS.md` для найденных проблем.
    
-   Смотрите `TESTCASES.md` для сценариев тестирования.

# Инструкция по запуску автотестов

1. **Скачайте проект**  
   Склонируйте репозиторий или скачайте ZIP-архив и распакуйте его

2. **Проверьте Python**  
   Убедитесь, что на компьютере установлен Python. Выполните в терминале:
   
		python --version

Если Python не установлен, скачайте с официального сайта и при установке отметьте галочку "Add Python.exe to PATH".

3.  **Перейдите в папку проекта**  
    В терминале перейдите в директорию проекта:

	    cd /путь/до/папки/Task 2
    
4.  **Установите зависимости**  
    Установите необходимые библиотеки:

		pip install -r requirements.txt
    
    Если команда не работает, попробуйте:

		pip3 install -r requirements.txt
    
5.  **Запустите тесты**  
    Выполните команду для запуска всех тестов:

	    pytest tests/ -v
