# Тестовое задание для QA-инженера

## О проекте

Проект содержит два задания по тестированию:
- **Task 1** - Поиск багов на скриншоте.
- **Task 2** - Написание автотестов для API.

## Структура проекта

    Avito_api_tests/  
    ├── Task1/ # Задание 1
    ├── Bugs.md # Описание найденных багов.  
        └── Task1_Screenshot.png # Скриншот для анализа.  
    ├── Task2/ # Задание 2.1
        ├── tests/ # Автотесты  
            ├── test_create.py # Тесты создания объявлений.
            ├── test_get.py # Тесты получения объявлений.
            ├── test_statistics.py # Тесты статистики.
            ├── test_delete.py # Тесты удаления объявлений.
            └── conftest.py # Конфигурация тестов.
        ├── utils/  # Утилиты
            └── API_client.py # API клиент.
        ├── BUGS.md # Найденные баги в API.
        ├── TESTCASES.md # Тест-кейсы.
        ├── README.md # Инструкция по запуску ко второму заданию.
        └── requirements.txt # Зависимости.
    ├── README.md #Этот файл.
    └── .gitignore

## Документация

-   `Task1/Bugs.md` - баг-репорты по скришоту.
    
-   `Task2/BUGS.md` - баг-репорты по API.
    
-   `Task2/TESTCASES.md` - тест-кейсы API.
    
-   `Task2/README.md` - инструкция по запуску автотестов.
